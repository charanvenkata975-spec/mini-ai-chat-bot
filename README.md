# MINI AI ∞

A high-performance local LLM interface built with:

- Node.js
- Express
- Ollama
- Streaming architecture
- Adaptive UI engine

## Features

- Real-time streaming responses
- Token-buffer optimized rendering
- Offline detection & retry logic
- Telemetry batching
- Session persistence
- Voice input & TTS
- Performance-aware UI design

## Tech Stack

Frontend:
- Vanilla JS
- Streaming engine
- Custom markdown parser
- Glassmorphism UI

Backend:
- Express proxy
- Ollama integration
- Vision-ready architecture
- Streaming pass-through

## Run Locally

1. Install dependencies

2. Start Ollama

3. Start server

4. Open

http://localhost:3000
